# Director Library

One file per director. Each file is a **director-mind block** that the app injects into the
`DIRECTOR_PROFILE` slot of `prompts/script_generator.system.md` when the user picks that director
from the dropdown. If no director is chosen, the base default ("versatile master") stays.

## How it composes
- **Director = the *how*** — voice, worldview, character architecture, dialogue texture, scene
  grammar, ending logic. The film should feel like it came out of this mind.
- **Mood = the *register*** — the genre/emotional key (see `prompts/moods/`).
- The base prompt's rule: *tell this mood's story through this director's mind.* A real director
  keeps their voice in any genre, so e.g. "Hirani-does-horror" or "Bhansali-does-comedy" resolves
  cleanly. Director leads the voice; mood leads the register.

## Template (every file follows it)
Core belief · What people are · Stories that pull you · Protagonist (wants vs needs) · Character
architecture · Dialogue DNA · Scene & structure · Pacing · Emotional technique · Visual grammar ·
Ending philosophy · Recurring obsessions · Blind spots to balance · In one breath (the directive).

## Usage / originality note
These are public "writing-engine" reconstructions built from interviews and observable patterns —
they are tools to emulate a director's **craft sensibility and instincts**, not their actual words,
and not a license to copy any specific film or scene. Aim for disciplined originality in their
spirit, never mimicry of a copyrighted work.
