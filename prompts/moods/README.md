# Mood Library

One file per mood. Each file is a **register block** the app injects into the `MOOD_DIRECTION`
slot of `prompts/script_generator.system.md` when the user picks that mood from the dropdown.

## Authority
**Mood is fully authoritative over register.** When a mood is selected, it governs pace, language,
imagery, escalation, and climax — even if the chosen director would naturally lean elsewhere. The
director still supplies the *voice/how*; the mood decides *what kind of film* it is. (Base rule:
"tell this mood's story through this director's mind," with the mood leading the register.)

## Default
If the user selects no mood, the default register is **Maximum Drama** (`maximum-drama.md`) — the
assignment's signature absurd, over-engineered register (small problem → massive drama). The moods
below are the explicit dropdown choices.
